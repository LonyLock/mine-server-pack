# Mine Server Pack

Готовый клиентский модпак для нашего Minecraft-сервера: **Fabric 26.2 + Sodium + Iris + шейдеры Complementary Reimagined** (включены из коробки).

Адрес сервера: **88.204.150.46**

## Установка — способ 1: один exe (самый простой)

1. Скачай `MineServerPack-Setup.exe` со [страницы релизов](../../releases/latest)
2. Запусти его (нужен установленный [официальный лаунчер](https://www.minecraft.net))
3. Открой лаунчер — там появится профиль **Mine Server Pack** с модами, шейдерами и уже прописанным сервером

> Windows SmartScreen может предупредить о неизвестном издателе (файл не подписан
> сертификатом). Нажми «Подробнее» → «Выполнить в любом случае». Исходный код
> установщика открыт: [src/MineServerPackSetup.cs](src/MineServerPackSetup.cs).

## Установка — способ 2: .mrpack (Modrinth App / Prism)

1. Скачай последний `.mrpack` со [страницы релизов](../../releases/latest)
2. Установи [Modrinth App](https://modrinth.app) (или [Prism Launcher](https://prismlauncher.org))
3. Перетащи `.mrpack` в окно лаунчера (или «+» → «From file»)
4. Лаунчер сам скачает Minecraft 26.2, Fabric и все моды
5. Запусти профиль и подключайся к `88.204.150.46`

## Что внутри

| Мод | Зачем |
|---|---|
| [Sodium](https://modrinth.com/mod/sodium) | Ускоряет рендер в 2–4 раза |
| [Iris](https://modrinth.com/mod/iris) | Поддержка шейдеров |
| [Complementary Reimagined](https://modrinth.com/shader/complementary-reimagined) | Сам шейдерпак (уже включён) |
| [Fabric API](https://modrinth.com/mod/fabric-api) | Базовая библиотека |
| [Lithium](https://modrinth.com/mod/lithium) | Оптимизация логики игры |

Шейдеры включены сразу. Отключить/поменять: `Настройки → Настройки графики → Пакеты шейдеров`.

Сам модпак не содержит файлов игры и модов — только ссылки на официальные загрузки с Modrinth, лаунчер скачивает всё сам.

## Можно ли зайти без модпака?

Да — сервер принимает и чистые (ванильные) клиенты версии 26.2.
Модпак нужен только для шейдеров и производительности.

## Для мейнтейнера: обновление пака

```
python scripts/build_mrpack.py 1.1.0   # собрать свежий .mrpack
scripts/build_exe.ps1                  # пересобрать установщик (нужно редко)
```

Скрипт запросит у Modrinth свежие версии модов под MC 26.2, соберёт
`Mine-Server-Pack-<версия>.mrpack` и его останется приложить к новому релизу.
Установщик пересобирать при обновлении модов не нужно — он всегда скачивает
последний релиз и берёт список модов из него.
